## Service Detection

```bash
nmap -p3128 -sC -sV -Pn TARGET
```

**Example**

```bash
PORT     STATE SERVICE    VERSION
3128/tcp open  http-proxy Squid http proxy 4.14
|_http-server-header: squid/4.14
|_http-title: ERROR: The requested URL could not be retrieved
```
