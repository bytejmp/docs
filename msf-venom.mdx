## Basic Syntax

```bash
msfvenom -p <PAYLOAD> LHOST=<IP> LPORT=<PORT> -f <FORMAT> -o <FILE>
```

### Common Options

| Option       | Description       |
| ------------ | ----------------- |
| `-p`         | Payload           |
| `LHOST`      | Attacker IP       |
| `LPORT`      | Listening port    |
| `-f`         | Output format     |
| `-o`         | Output file       |
| `-e`         | Encoder           |
| `-i`         | Encode iterations |
| `-b`         | Bad chars         |
| `--platform` | Force platform    |
| `-a`         | Architecture      |
